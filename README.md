## 📌 Write a Message Web Application

Displays texts from the user's input after submission.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Displays the user's message in green after submission of the message.
* And displays red if the input field is empty.
* Provides a fun and engaging user experience.
